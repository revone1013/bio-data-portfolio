# PBMC Single-Cell RNA-seq Analysis

This folder contains the single-cell transcriptomic analysis of peripheral blood mononuclear cells (PBMCs).  
The project explores clustering, cell-type annotation, and marker gene identification using R and Seurat.

## Files
- `pbmc_analysis.Rmd`: R Markdown file containing the analysis pipeline.
- `pbmc_analysis.html`: Rendered report output.

## Tools and Packages
- R 4.x
- Seurat, dplyr, ggplot2
- SingleR (for cell type annotation)

## Highlights
- Dimension reduction (PCA, UMAP)
- Marker gene discovery
- Comparison between clusters
- Visualization with ggplot2
